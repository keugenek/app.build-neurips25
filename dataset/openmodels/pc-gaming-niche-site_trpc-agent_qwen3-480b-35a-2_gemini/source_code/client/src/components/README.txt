Budget PC Gaming Hub - Component Documentation

This directory contains all the UI components for the budget PC gaming peripheral review site.

Main Components:
- LandingPage.tsx: Homepage with featured content
- ProductCard.tsx: Individual product display
- ArticleCard.tsx: Individual article display
- CategoryFilter.tsx: Product category filtering
- AdminPanel.tsx: Content management interface
- FeaturedProduct.tsx: Special product display
- ArticlePreview.tsx: Compact article display

The site is organized around:
1. Product categories (mice, keyboards, etc.)
2. Product listings with reviews
3. Published articles and guides
4. Admin interface for content management
